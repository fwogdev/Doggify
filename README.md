## Welcome to Doggify
** Doggify is a Open-Source Chromium BookMark that create a image of a dog and over-lays it over your screen.

Doggify is completely made is JavaScript (.js), and doesn't have plans to update it the future (for obvious reasons).

# Installation
** First go to the Doggify_Source.js in this repository, next open the file and copy the code inside.

** Then, at the top of your screen, where your book marks are located, right click the bar and press "Add Page..."
